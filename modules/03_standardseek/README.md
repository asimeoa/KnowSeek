# 📏 StandardSeek Module
**Automated compliance and norm management.**

### Purpose
StandardSeek ensures that every design follows the required industry standards and OEM-specific norms (e.g., DIN, ISO, or specific VW/BMW standards).

### Key Features
* **Norm Monitoring:** Get notified when a relevant standard changes.
* **Automated Check:** Upload a document and let the AI verify if it meets the latest requirements.
* **Legacy Check:** Scan old projects to see if they still comply with modern regulations.
